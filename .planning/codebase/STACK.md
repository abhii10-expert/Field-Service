# Technology Stack

## Core Frameworks
- **Salesforce Lightning Web Components (LWC)**: Main UI framework for the `cpeAiAssistant`.
- **Apex**: Backend logic, controller actions, and Agentforce Invocable methods.
- **Agentforce (Einstein)**: AI reasoning engine for FAQ, Troubleshooting, and Inventory management.

## Languages
- **JavaScript (ES6+)**: Frontend logic, event handling, and voice integration (Web Speech API).
- **Apex (Java-like)**: Server-side logic and database interactions.
- **HTML/CSS**: UI structure and premium glassmorphism styling.

## Key APIs & Libraries
- **ConnectAPI (AgentService)**: Integration point for the Agentforce LLM.
- **Web Speech API**: Native browser support for voice-to-text functionality.
- **Salesforce Field Service (SFS)**: Core data model (ServiceResource, ProductItem, Location).

## Deployment & Tooling
- **SFDX / Salesforce CLI**: Metadata deployment and project management.
- **GSD (Get Shit Done)**: Project planning and codebase mapping toolkit.
